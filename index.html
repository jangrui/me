    项目

    广铁文库系统重构

    项目描述

    因现有文库系统负载较高，访问迟钝，无法满足正常办公，对此重构升级。

    项目要求

    实现高并发，高可用，高性能。

    任务描述

  • 利用 PXE + Kickstart 批量安装 CentOS7 以及 Docker 环境；
  • 构建文库系统 docker 镜像；
  • 利用 docker 部署文库系统及依赖:
  • haproxy 集群：2节点；
  • 文库系统集群：3节点；
  • mycat 集群：2 节点；
  • mysql mha 集群：1主1备2从节点；
  • minio 集群：4节点；
  • elasticsearch 集群：1主3从节点；
  • kibana：1节点；
  • zookeeper：1节点；
  • kafka 集群：1主3从节点；
  • logstash 集群：3节点；
  • 利用 LVS + keepalived 实现高可用热备方案；

    • 项目总结

    部署此项目前未接触过 ELK ，边扒文档边部署，花费不少力气，也因此对日志分析产生浓厚兴趣；通过此项目，加深了对 mysql 集群的理解，以及各服务之间相互衔接有进一步认识。
